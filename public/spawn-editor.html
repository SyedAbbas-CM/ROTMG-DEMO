How to Use

This editor helps you visually configure enemy spawns for each world in your game.

Quick Start:

  1. Select a world from the dropdown
  2. Click on an existing spawn to edit it, or click "Add New Spawn"
  3. Choose enemy type and set coordinates
  4. Click "Save Spawn" to update
  5. Use "Export Config" to generate the code
  6. Copy the generated code to config/world-spawns.js
  7. Restart the server to apply changes

Available Enemy Types:

  • Goblin - Basic enemy, good for starting areas (30 HP)
  • Orc - Tougher enemy that retreats when wounded (50 HP)
  • Red Demon - Mini-boss with spread shots (350 HP)
  • Hyper Demon - Super boss for endgame (50000 HP)
  • Charging Shooter - Mobile enemy that charges and shoots (80 HP)

Tips:

  • Place spawn coordinates based on your map size
  • Add comments to spawns for better organization
  • Group enemies by area (forest, mountains, etc.)
  • Leave safe zones (like Nexus) with no enemies
  • Test spawns in-game before finalizing
